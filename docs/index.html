我有问题要反馈，怎么对项目做出贡献？

方法很简单，你并不需要懂得如何编写网页代码，你可以在Github仓库 (英文版)找到说明，或者直接去反馈bug (英文版) (中文版)，我们会尽快修复汇报的问题。

我可以追踪多个角色吗？

可以，用选项页签的档案选择器设置多个不同的档案。

列表追踪状态是怎么保存的？

列表追踪状态数据保存在浏览器本地存储里。小心！清除浏览器缓存也会清空你的数据。
